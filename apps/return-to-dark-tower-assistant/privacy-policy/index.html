Privacy Policy

I respect your privacy. For that reason, Return to Dark Tower Assistant doesn't collect any personally identifiable information. I use TelemetryDeck to create basic usage data in order to keep an eye on what iOS versions are being used, etc, but this is never linked to any user data. The app does not require or use an internet connection.

Return to Dark Tower Assistant contains no advertising or in-app purchases.
