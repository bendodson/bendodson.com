Keep your hero board under control

Return to Dark Tower is brilliant, but a busy hero board can be hard to keep straight. Return to Dark Tower Assistant helps track phases, actions, cards, Advantages, and hero board state so you can spend less time checking rules and more time playing.

It supports 1-4 players, works beautifully in iPad Slide Over above the official companion app, and includes support for the Alliances and Covenant expansions.
